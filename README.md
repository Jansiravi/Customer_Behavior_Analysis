

# 📊 Customer Shopping Behavior – Data Analysis Project

## 📌 Overview

This project focuses on analyzing **customer shopping behavior** using data analysis techniques. The goal is to extract meaningful insights from the dataset by performing **data cleaning, exploratory data analysis (EDA), SQL-based business queries, and interactive dashboard creation**.

The project demonstrates the complete **data analytics workflow**, starting from raw data processing in **Python**, performing **business analysis using SQL**, and presenting insights through a **Power BI dashboard**.

---

## 📂 Dataset

The dataset contains information about customer shopping behavior, including:

* Customer demographics (Age, Gender, Location)
* Product details (Item Purchased, Category, Size, Color)
* Purchase details (Purchase Amount, Frequency)
* Payment methods
* Discounts and promo codes
* Customer ratings and previous purchases

The dataset includes **3,900 records and multiple features** related to customer purchasing patterns.

## 🔎 Project Workflow

### 1️⃣ Data Loading

* Dataset was imported from **Excel** into **Python using Pandas**.
* Initial inspection of dataset structure and data types.

### 2️⃣ Data Cleaning

Key preprocessing steps included:

* Handling missing values
* Checking duplicates
* Correcting data types
* Data formatting and consistency checks

### 3️⃣ Exploratory Data Analysis (EDA)

EDA was performed to understand patterns and relationships in the data, including:

* Customer age distribution
* Purchase trends by product category
* Seasonal purchasing behavior
* Impact of discounts and promo codes
* Popular payment methods

Visualization libraries such as **Matplotlib and Seaborn** were used.

---

## 💼 Business Questions Solved Using MySQL

Some key business questions answered in this project include:

* Which **product categories generate the highest revenue**?
* Which **locations have the highest number of purchases**?
* What is the **average purchase amount by gender**?
* Which **payment methods are most commonly used**?
* How does **purchase frequency affect customer spending**?

SQL queries were written and executed in **MySQL** to derive these insights.

---

## 📊 Power BI Dashboard

An interactive **Power BI dashboard** was created to visualize the key insights.

### Dashboard Highlights

* Total Sales Overview
* Purchase Distribution by Category
* Customer Demographics Analysis
* Payment Method Usage
* Seasonal Purchase Trends

The dashboard helps stakeholders **quickly understand customer behavior and purchasing patterns**.

---

## 📈 Key Insights

Some insights identified from the analysis include:

* Certain product categories generate **higher customer engagement and revenue**.
* **Seasonal trends** influence purchasing behavior.
* **Discounts and promo codes** impact purchase decisions.
* Some **payment methods are significantly more popular** among customers.
